1.3 Elementos Principales

Elementos principales del computador

  • CPU (Unidad Central de Procesamiento): el cerebro de la computadora
  • Memoria RAM: almacenamiento temporal para procesos activos
  • Almacenamiento: HDD, SSD para guardar datos
  • Tarjeta gráfica (GPU): procesamiento visual
  • Placa base: conecta todos los componentes
  • Fuente de poder: suministra energía

1.4 Unidades de Medición

Unidades para medir almacenamiento, procesamiento y transmisión

  • Bit y Byte: unidades básicas de información
  • KB, MB, GB, TB: unidades de almacenamiento
  • Hertz (Hz): velocidad de procesamiento
  • bps (bits por segundo): velocidad de transmisión
Unidad Descripción

1.10 Curiosidades

Curiosidades sobre la computadora

  • El primer "bug" fue literalmente un insecto (una polilla) dentro de un computador Harvard Mark II.
  • El ENIAC ocupaba una habitación entera y consumía tanta energía como una pequeña ciudad.
  • Los primeros computadores usaban tarjetas perforadas para la entrada de datos.
  • Hoy en día, la potencia de un smartphone supera con creces a la de las primeras computadoras.
  • El término "byte" fue acuñado por Werner Buchholz en 1956.
