Requisitos Técnicos

Hardware Requerido

  • Procesador: 64-bit con 2 núcleos (1GHz+)

Paso 0: Verificación del Disco Duro

Antes de instalar cualquier sistema operativo, es esencial verificar el estado de tu disco duro para prevenir problemas futuros.

Verificación en Windows

  1. Abre el Explorador de archivos
  2. Haz clic derecho en tu disco principal (C:)
  3. Selecciona "Propiedades" > "Herramientas"
  4. En "Comprobación de errores", haz clic en "Comprobar"
  5. Selecciona "Examinar unidad"
  6. Espera a que se complete el análisis
chkdsk C: /f /r /x

Verificación en Linux

  1. Abre la Terminal (Ctrl+Alt+T)
  2. Instala smartmontools:
    sudo apt install smartmontools
  3. Verifica el estado del disco:
    sudo smartctl -a /dev/sda
  4. Busca la línea "SMART overall-health self-assessment test result"

¡Atención! Si tu disco muestra errores o advertencias, considera reemplazarlo antes de continuar con la instalación.

Paso 1: Instalación Dual Boot

Instalación detallada paso a paso para principiantes con imágenes de referencia:

Preparación

  1. Haz copia de seguridad de todos tus datos importantes
  2. Descarga las ISO de Windows 11 y Ubuntu
  3. Crea USBs de instalación con Rufus (selecciona GPT para UEFI)
  4. Accede a la BIOS (generalmente pulsando F2, F10 o DEL al iniciar)
  5. Desactiva Secure Boot
  6. Configura el orden de arranque para que USB sea primero
Configuración de la BIOS para desactivar Secure Boot

Instalación de Windows 11

  1. Arranca desde el USB de Windows
  2. Selecciona idioma y distribución de teclado
  3. En "Tipo de instalación", elige "Personalizada"
  4. Elimina todas las particiones existentes
  5. Crea nuevas particiones:
    • EFI (500MB)
    • Particion para Windows (100GB+)
  6. Selecciona la partición principal y haz clic en "Siguiente"
  7. Sigue las instrucciones para completar la instalación
Creación de particiones durante la instalación de Windows

Instalación de Ubuntu

  1. Arranca desde el USB de Ubuntu
  2. Selecciona "Instalar Ubuntu"
  3. Elige tu idioma y distribución de teclado
  4. En "Tipo de instalación", selecciona "Instalar junto a Windows Boot Manager"
  5. Asigna espacio para Ubuntu (50GB+)
  6. Crea usuario y contraseña
  7. Espera a que se complete la instalación
  8. Reinicia el sistema
Seleccionar "Instalar junto a Windows"

Paso 2: Instalación de Controladores

Los controladores son esenciales para que todo el hardware funcione correctamente en ambos sistemas operativos.

Controladores en Windows 11

  1. Actualiza Windows a través de Windows Update
  2. Verifica el Administrador de dispositivos para dispositivos sin controlador
  3. Visita el sitio web del fabricante para descargar controladores específicos:
    • Tarjeta gráfica (NVIDIA/AMD/Intel)
    • Tarjeta de red (WiFi/Ethernet)
    • Audio
  4. Instala los controladores descargados

Controladores en Ubuntu

  1. Abre "Software y Actualizaciones"
  2. Ve a la pestaña "Controladores Adicionales"
  3. Selecciona los controladores recomendados para tu hardware
  4. Haz clic en "Aplicar cambios"
  5. Para controladores NVIDIA:
    sudo ubuntu-drivers autoinstall
  6. Reinicia el sistema después de instalar

Controladores Esenciales

Componente Windows Ubuntu Importancia
Gráficos Controlador del fabricante Controlador propietario o Nouveau Crítica
Red Actualización automática Generalmente incluido Alta
Audio Realtek/Intel HD Audio ALSA/PulseAudio Alta
Impresora Controlador específico CUPS + controlador Media

Paso 3: Instalar Microsoft Office en Windows

Método Oficial

  1. Inicia sesión en office.com
  2. Haz clic en "Instalar Office"
  3. Descarga el instalador (OfficeSetup.exe)
  4. Ejecuta el instalador y espera la descarga
  5. Inicia sesión con tu cuenta Microsoft
  6. Personaliza la instalación si es necesario
  7. Espera a que se complete la instalación (10-20 min)

Versiones disponibles: Office 365 (suscripción) o Office 2021 (compra única).

Proceso de instalación de Office

Configuración Inicial

  1. Abre cualquier aplicación de Office (Word, Excel)
  2. Acepta los términos de licencia
  3. Personaliza tu experiencia (opcional)
  4. Conecta OneDrive si lo deseas
  5. ¡Listo para usar!

Office se integra automáticamente con Windows 11. Puedes abrir documentos desde el Explorador de archivos.

Paso 4: Alternativas de Office en Ubuntu

Aplicación Descripción Compatibilidad Método de Instalación
LibreOffice Suite ofimática gratuita y de código abierto Excelente con ODF, buena con formatos Microsoft Preinstalado en Ubuntu
OnlyOffice Interfaz similar a MS Office con alta compatibilidad Muy buena con formatos DOCX, XLSX Snap Store o descarga directa
WPS Office Alternativa gratuita con interfaz casi idéntica Excelente compatibilidad con MS Office Descarga desde sitio oficial

Instalar OnlyOffice

  1. Abre la terminal (Ctrl+Alt+T)
  2. Instala mediante Snap:
    sudo snap install onlyoffice-desktopeditors
  3. O descarga desde onlyoffice.com
  4. Ejecuta el archivo .deb descargado
  5. Instala usando Centro de Software de Ubuntu

Paso 5: Google Drive en Windows

  1. Descarga "Drive para computadora" desde google.com/drive
  2. Ejecuta el instalador
  3. Inicia sesión con tu cuenta de Google
  4. Elige las carpetas a sincronizar
  5. Configura las preferencias de sincronización
  6. Espera a que se complete la sincronización inicial

Google Drive aparecerá como una unidad en el Explorador de archivos, manteniendo todos tus archivos accesibles localmente.

Google Drive en el Explorador de Windows

Configuración Avanzada

  • Sincronización selectiva: Elige qué carpetas mantener en tu equipo
  • Copias de seguridad: Configura copia de seguridad de carpetas importantes
  • Acceso sin conexión: Marca archivos para acceso offline
  • Ancho de banda: Limita el uso de red si es necesario

Paso 6: Google Drive en Ubuntu

Método 1: Integración con Archivos

  1. Abre "Configuración del sistema"
  2. Ve a "Cuentas en línea"
  3. Haz clic en "Añadir cuenta" y elige Google
  4. Inicia sesión con tu cuenta de Google
  5. Activa "Archivos" para la integración
  6. Abre el administrador de archivos
  7. Google Drive aparecerá en la barra lateral
Google Drive en el administrador de archivos

Método 2: Aplicación ODrive

  1. Abre la terminal (Ctrl+Alt+T)
  2. Instala ODrive:
    sudo snap install odrive
  3. Ejecuta ODrive desde el menú de aplicaciones
  4. Haz clic en "Añadir cuenta" y elige Google
  5. Inicia sesión y autoriza el acceso
  6. Selecciona las carpetas a sincronizar
Interfaz de ODrive para Ubuntu

Configuración Final y Consejos

Configuración Windows

  • Configura OneDrive para sincronización automática
  • Vincula tu cuenta Microsoft para activar Office
  • Personaliza la barra de acceso rápido en Office

Configuración Ubuntu

  • Establece OnlyOffice como predeterminado para documentos
  • Configura sincronización selectiva en Google Drive
  • Instala fuentes Microsoft para mejor compatibilidad:
    sudo apt install ttf-mscorefonts-installer

Consejos de Sincronización

  • Usa nombres de archivo compatibles (sin caracteres especiales)
  • Evita trabajar en el mismo archivo simultáneamente
  • Revisa siempre las versiones después de sincronizar
